# Amber 花藝

一頁式品牌網站，包含首頁、品牌介紹、商品展示、品牌故事與聯絡方式。

## GitHub Pages 部署方式

1. 到 GitHub 建立新 Repository，例如 `amber-floral`.
2. 將本資料夾內的檔案上傳到 Repository 根目錄：
   - `index.html`
   - `styles.css`
   - `.nojekyll`
   - `assets/amber-floral-hero.png`
3. 進入 Repository 的 `Settings`。
4. 左側選擇 `Pages`。
5. `Build and deployment` 選擇：
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
6. 按下 `Save`。
7. 等待 GitHub Pages 完成部署後，即可取得網站網址。

部署後網址通常會是：

```text
https://你的帳號.github.io/amber-floral/
```
